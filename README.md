## ğŸš€ Getting Started

Want to run this project locally? Follow these simple steps:

### Prerequisites

-   **Node.js** (v18 or higher)
-   **npm** or **yarn**

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/abhishek0112cs221008/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  That's it! Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
